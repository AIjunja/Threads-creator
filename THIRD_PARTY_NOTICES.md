# Third-Party Notices

## Moneygraphy

This project supports Moneygraphy font files in `assets/fonts` so the desktop UI can render consistently when the user has downloaded the font separately.

Moneygraphy is free for personal and commercial use, but public redistribution of the font files is prohibited according to third-party license summaries. For that reason, this public repository does not include the font files.

Users who want the Moneygraphy UI style should download the font from the official source and place it in `assets/fonts`.
